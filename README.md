# 💻 Персональне портфоліо Frontend-розробниці

Ласкаво просимо до репозиторію мого особистого сайту-портфоліо! Це адаптивний, сучасний та швидкий вебсайт, виконаний у глибоких смарагдових тонах. Він створений для демонстрації моїх навичок, бекграунду та виконаних проєктів.

## 🚀 Технологічний стек
* **HTML5** — семантична розмітка та SEO-оптимізація.
* **CSS3** — адаптивний дизайн (Flexbox, CSS Grid), кастомні змінні (Variables) та інтерактивні ховер-ефекти.
* **JavaScript** — логіка інтерфейсу, динамічні елементи.

## 📁 Проєкти, представлені на сайті:
1. **Healthy Baby** — багатосторінковий сайт про прикорм із калькулятором персонального графіка щеплень (чистий JavaScript).
2. **Latte Planner** — Progressive Web App (PWA) застосунок-блокнот із локальною базою даних у `LocalStorage`.
3. **ЮліяDigital** — комерційний Landing Page для продажу цифрових товарів з акцентом на UX/UI та високу конверсію.

## ⚙️ Особливості сайту:
* Повна адаптивність під мобільні телефони, планшети та монітори (Mobile First).
* Швидке завантаження завдяки відсутності важких фреймворків чи конструкторів.
* Охайний, структурований код, готовий до подальшої інтеграції з WordPress.
